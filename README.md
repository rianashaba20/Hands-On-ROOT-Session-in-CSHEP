# Hands-On-ROOT-Session-in-CSHEP
Module 3 of the course CSHEP taken during the third IMAPP semester at University of Bologna. The module focuses on selected statistical analysis cases in High Energy Physics, using RooFit and RooStats as the main tools.

## Content
- Assignment: [MINOS Neutrino Experiment](Assignments/Exercise11_Ad.ipynb)
- Assignment: [LHCb Breit-Wigner Model](Assignments/Exercise12_2.ipynb)
- Assignment: [LHCb - First Observation of the Rare Decay $B \rightarrow pp$](Assignments/ExerciseA.ipynb)
- Assignment: [ATLAS Measurement of the Higgs Boson Mass](Assignments/ExerciseATLAS.ipynb)
- Assignment: [OPERA Neutrino Oscillations](Assignments/ExerciseOPERA.ipynb)
- Assignment: [Maximum Likelihood Fit to an binned dataset](Assignments/Exercise2021_2.ipynb)
- Assignment: [Maximum Likelihood Fit to an ubinned dataset](Assignments/Exercise2021_1.ipynb)
- Assignment: [Non-extended Composite Model: Signal + Background](Assignments/Exercise13_1.ipynb)
- Assignment: [Extended Composite Model: Signal + Background](Assignments/Exercise13_2.ipynb)
